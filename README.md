# Timer

### Setup

Make sure [bower](http://bower.io/) is installed in your local.

```bash
$ git clone git@github.com:HouCoder/Timer.git
$ cd Timer
$ bower install
$ python -m SimpleHTTPServer 8080

```

Then open [http://127.0.0.1:8080](http://127.0.0.1:8080) in your browser :tada:.
